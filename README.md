## 🌟 项目简介

**智绘童心** 是一款现代化的 Web 应用，旨在简化简历制作过程。它结合了 AI 技术与直观的 UI 设计，让家长和学生能够快速生成包含封面、基本信息、素质报告、证书荣誉、自荐信等全方位内容的精美简历。

## ✨ 核心功能

- **🤖 AI 智能辅助**：集成 Google Gemini API，自动优化自荐信内容，提供创意灵感。
- **📋 全模块化设计**：
  - **精美封面/封底**：支持自定义背景、头像框及个性化寄语。
  - **基本档案**：清晰展示个人信息、联系方式及学业成绩表。
  - **素质报告**：支持上传并分页展示综合素质评价。
  - **荣誉证书**：智能排版系统，自动将证书分类展示（每页 4 张）。
  - **兴趣特长**：图文混排模式，生动展现孩子的多彩生活。
  - **社会实践**：记录丰富的实践经历与感悟。
  - **稿纸自荐信**：独创的红框稿纸布局，模拟手写感，支持 AI 一键润色。
- **👁️ 实时排版预览**：
  - **标准 A4 比例**：完全模拟真实打印效果。
  - **双模式预览**：支持“单列排版”与“三列并排（全景模式）”，方便全局调整。
- **🎨 高度定制化**：
  - 多套精选主题色，一键切换整体风格。
  - 丰富的头像形状（圆形、方形、星形等）与装饰框。
- **🖨️ 专业级导出**：
  - 针对打印优化的 CSS 样式。
  - 支持直接保存为高清 PDF。

## 🛠️ 技术栈

- **前端框架**：React 19 + TypeScript
- **样式处理**：Tailwind CSS (响应式、动态主题)
- **图标库**：Lucide React
- **AI 集成**：Google Gemini SDK
- **构建工具**：Vite
- **工具库**：html2pdf.js, react-image-crop

## 🚀 快速开始

### 环境要求
- [Node.js](https://nodejs.org/) (建议最新 LTS 版本)

### 安装与运行

1. **克隆项目**
   ```bash
   git clone [repository-url]
   cd 智绘童心---ai-简历生成器
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **配置 API Key**
   在根目录下创建 `.env.local` 文件，并添加你的 Gemini API 密钥：
   ```env
   VITE_GEMINI_API_KEY=你的_GEMINI_API_KEY
   ```

4. **启动开发服务器**
   ```bash
   npm run dev
   ```
   访问 `http://localhost:3001` 即可开始使用。

## 💡 开发说明

### 主题系统
项目采用 CSS 变量与 Tailwind CSS 结合的方式实现动态主题。所有主题配置位于 `types.ts` 的 `ThemeType` 中，并在 `ResumePreview.tsx` 的 `getThemeStyles` 函数中转换为实际的 CSS 变量。

### A4 排版逻辑
- **尺寸控制**：预览页面固定为 `210mm x 297mm` (A4 标准)，并使用 `origin-top-left` 缩放实现不同屏幕尺寸下的完美适配。
- **分页逻辑**：针对素质报告、荣誉证书等内容，系统会自动进行数据分块（Chunking），确保内容在跨页时不会截断。
- **打印优化**：使用 `@media print` 媒体查询，在打印时自动隐藏编辑器 UI，并重置缩放比例。

### AI 功能集成
AI 润色功能通过 `services/geminiService.ts` 调用 Google Gemini 1.5 Flash 模型。为了保证稳定性，建议在 `.env.local` 中配置有效的 API Key。

## 📖 使用指南

1. **填写信息**：在左侧边栏按步骤输入基本信息、上传照片及证书。
2. **主题定制**：点击顶部的颜色气泡，实时切换简历配色。
3. **AI 创作**：在“自荐信”模块，输入关键词后点击“AI 辅助生成”，获取高质量初稿。
4. **排版调整**：利用预览区的“三列”模式检查全书连贯性，调整图片顺序。
5. **导出 PDF**：点击右上角的“导出 PDF”，在打印设置中选择“另存为 PDF”，边距设为“无”即可。
